Gérer les bulletins de notes avec Simpala

12 février 2018

Grace aux applications Simpala Console et Simpala Gradebook, le responsable pédagogique d’un établissement dispose de tous les outils requis pour générer des bulletins de très bonne qualité.
Afin de vous guider à travers les différentes étapes, nous vous proposons le parcours suivant :

  1. créer les groupes d’élèves ;
  2. déclarer les classes ouvertes pour l’année en cours ;
  3. inscrire les élèves dans leurs classes respectives ;
  4. créer un compte pour chaque enseignant ;
  5. déclarer les matières dispensées dans l'établissement ;
  6. déclarer les entités qui vous permettront de regrouper les matières dans vos bulletins ;
  7. définir les méthodes de calcul des moyennes séquentielles et trimestrielles d'une classe ;
  8. déclarer les matières dispensées dans une classe ;
  9. définir le type de bulletin, définir les entêtes et textes des signatures ;
  10. insérer les notes d'une séquence ;
  11. personnaliser les appréciations du travail dans chaque matière ;
  12. enregistrer les sanctions disciplinaires ;
  13. générer les bulletins.

Pour aller plus loin 

  • définir l'ordre d'affichage des groupes de matières sur le bulletin ;
  • envoyer les resultats scolaires par sms
