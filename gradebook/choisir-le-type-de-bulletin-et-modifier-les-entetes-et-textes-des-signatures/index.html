Choisir le type de bulletin et modifier les entêtes et textes des signatures

09 mars 2018

NB: Cette action n’est autorisée que si vous avez la permission de configurer les paramètres de l’application Gradebook (en savoir plus).

Pour ce faire :

  • aller à la section Gradebook > Paramètres ;

  • dans l’onglet Bulletins de notes :

    • cliquer sur le bouton Modifier si vous voulez modifier les paramètres du bulletin ;
    • cliquer sur le bouton Ajouter pour ajouter un type de bulletin ;

  • ensuite, éditer les paramètres généraux du bulletin et personnaliser les textes d’entête et des signatures ;

  • cliquer sur le bouton Enregistrer pour terminer.

Exemples de textes d’entête

Vous trouverez ci dessous quelques exemples de captures d’écran suivis du texte d’entête correspondant. Vous pouvez vous en inspirer, copier-coller et personnaliser uniquement les mots sans toucher aux balises HTML

il faut saisir le texte ci-dessous en respectant les consignes citées plus haut :

<i>Peace - Work - Fatherland</i><br>
MINISTRY OF SECONDARY EDUCATION<br>
REGIONAL DELEGATION FOR CENTRE<br>
DEPARTMENTAL DELEGATION FOR MFOUNDI<br>
