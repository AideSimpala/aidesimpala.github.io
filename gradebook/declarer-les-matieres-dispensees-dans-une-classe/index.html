Déclarer les matières dispensées dans une classe

16 mars 2018

NB: Cette action n’est autorisée que si vous avez la permission de configurer les paramètres de l’application Gradebook (en savoir plus).

Pour ce faire :

  • aller à la section Gradebook > Paramètres ;
  • dans le menu déroulant Groupes d’apprenants, sélectionner la classe correspondante puis cliquer sur le bouton Ajouter une matière ;
  • dans la fenêtre modale qui s’ouvre :
    • sélectionner la matière (penser à la créer si elle ne figure pas dans la liste déroulante) ;
    • définir le coefficient de la matière ;
    • définir le groupe de matières (penser à le créer si il ne figure pas dans la liste déroulante) ;
    • rechercher et selectionner l’enseignant qui dispense la matière (penser à créer son compte si besoin) ;
  • cliquer sur le bouton Enregistrer pour terminer.
