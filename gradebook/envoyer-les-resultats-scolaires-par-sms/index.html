Envoyer les résultats scolaires par SMS

09 mars 2018

NB: Cette action n’est autorisée que si vous avez la permission d’Envoyer des notifications aux parents à partir de l’application Gradebook (en savoir plus).

Afin de faciliter la communication entre l’établissement et les parents d’élèves, il est possible d’envoyer les résultats scolaires (séquentiels, trimestriels, annuels) par SMS.
Pour garantir la livraison de tous les messages, il est nécessaire de :

  • s’assurer que l’établissement dispose d’un crédit suffisant dans la réserve Gradebook de l’application Simpala Alerts ;
  • s’assurer que les bulletins de notes aient été générés et vérifiés.

La procédure est la suivante :

  • aller à la section Gradebook > Bulletins ;
  • sélectionner la PÉRIODE ou un GROUPE D’APPRENANTS afin d’afficher la liste des bulletins correspondente ;
  • cliquer sur l’enveloppe ouverte afin de préparer et envoyer une première fois les notifications ;
  • si vous souhaitez les envoyer une n-ème fois cliquer sur l’enveloppe fermée pour effectuer un nouvel envoi.
