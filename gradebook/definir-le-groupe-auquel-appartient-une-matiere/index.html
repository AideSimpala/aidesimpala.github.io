Définir le groupe auquel appartient une matière enseignée dans une classe

09 mars 2018

NB: Cette action n’est possible que si vous avez la permission de configurer les paramètres de l’application Gradebook (en savoir plus).

Pour ce faire :

  1. aller à la section Gradebook > Paramètres ;
  2. cliquer sur l’onglet Groupes d’apprenants ;
  3. dans le menu déroulant qui s’affiche, sélectionner la classe correspondante :
    • cliquer sur le bouton Ajouter une matière, dans le cas de l’ajout de la matière dans une classe ;
    • cliquer sur le bouton du menu contextuel , dans le cas de l’édition de la matière ;
  4. remplir le formulaire qui s’affiche, puis cliquer sur le bouton Enregistrer.
