#Les trois choses à ne pas faire lorsqu’on se connecte sur un poste autre que le sien
Pour des mesures de sécurités, et de confidentialités, il est très important de prendre des précautions lorsqu’il s’agit d’informations personnelles ou de données personnelles. A cet effet, pour éviter des situations embêtantes, il faut :

  • Évitez d’enregistrer vos paramètres de connection dans un Navigateur. A savoir :
    • Son login
    • Son mot de pass
  • Evitez les outils d’enregistrement de cookies
  • Naviguez en mode privé
    La navigation privée , est une fonction complémentaire de plusieurs navigateurs web, permettant de naviguer sur le web sans que les données de navigation comme la partie historique ou les cookies soient conservées lors du prochain lancement du logiciel.
    Vous avez le choix entre plusieurs navigateurs pour vous connecter en navigation privée. A savoir :
    • Activer la navigation privée sur Internet Explorer
      • Cliquez sur l’icône représentant un engrenage, située en haut à droite de votre navigateur.
      • Dans le menu qui apparaît, sélectionnez Sécurité.
      • Entrez dans la rubrique Navigation InPrivate
        Pour en savoir plus, cliquez sur le lien ci dessous :
        Navigation privée sur Internet Explorer
    • Activer la navigation privée sur Firefox
      • Appuyez sur l’icône des onglets en haut de l’écran.
      • Appuyez sur l’icône de masque en haut de l’écran pour passer en mode navigation privée. …
      • Appuyez sur le bouton de nouvel onglet pour ouvrir un onglet privé.
        Pour en savoir plus, cliquez sur le lien ci dessous :
        Navigation privée sur Firefox
    • Activer la navigation privée sur Google Chrome
      • Windows, Linux et Chrome OS : appuyez sur Ctrl+shift+n.
      • Mac : appuyez sur ⌘+shift+n.
        Pour en savoir plus, cliquez sur le lien ci dessous :
        Navigation privée sur Chrome
    • Activer la navigation privée sur le téléphone
      • Lancez le navigateur Internet.
      • Touche l’icône Pages.
      • Touchez l’icône Incognito.
      • Une nouvelle page s’ouvre alors en navigation privée.
        Pour en savoir plus, cliquez sur le lien ci dessous :
        Navigation privée sur Téléphone
    • Activer la navigation privée sur iphone
      • Appuyez sur Applications dans la section Contenu autorisé du menu Restrictions.
      • Décochez les classes d’âge que vous souhaitez masquer.
      • Cochez l’option Ne pas autoriser les applications afin de masquer toutes les applications téléchargées.
        Pour plus d’informations, cliquez sur le lien en dessous :
        Navigation privée sur Iphone
    • Activer la navigation privée sur Safari
      • Lancez le navigateur web Safari.
      • Appuyez ⌘+N.
        Pour en savoir plus, cliquez sur le lien en dessous :
        Navigation privée sur Safari
    • Activer la navigation privée sur Opéra
      • Cliquez sur l’icône de Microsoft Edge
      • Cliquez sur les trois petits points, en haut à droite
      • Ensuite Cliquez sur Nouvelle fenêtre InPrivate
        Pour en savoir plus, cliquez sur le lien en dessous :
        Navigation privée sur Opéra
    • Activer la navigation privée sur Edge
      • Cliquez sur l’icône de Microsoft Edge
      • Cliquez sur les trois petits points, en haut à droite.
      • Ensuite Cliquez sur Nouvelle fenêtre InPrivate
        Pour en savoir plus, cliquez sur le lien en dessous :
        Navigation privée sur Edge
        NB pour se désactiver le mode de navigation privée, déconnectez vous simplement des pages ouvertes en mode privé, et ouvrez une nouvelle page
