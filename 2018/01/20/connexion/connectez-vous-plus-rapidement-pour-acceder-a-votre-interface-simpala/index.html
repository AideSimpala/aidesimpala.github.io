#Connectez vous plus rapidement pour accéder à votre interface Simpala
La plate forme Simpala offre la possibilité à ses utilisateurs de se logger en un seul clic ; et par conséquent de gagner en temps au niveau de l’interface de connexion. Pour celà, il faut au préalable avoir renseigner l’adresse de l’utilisateur comme “Email principale” :

  • Cliquez sur “Utilisateurs” dans le menu latéral gauche ;
    !Bouton de l’onglet utilisateur
  • Cliquez sur le bouton de présentation du profil de l’utilisateur choisit : !Bouton de présentation de profil d’un utilisateur ; par exemple l’utilisateur nommé “ARMSTRONG Sebastien”
    !Profil de l’utilisateur
    La fenêtre suivante s’ouvre :
    !Profil de l’utilisateur
  • Cliquez sur le bouton “Modifier” et saisissez l’email principale :
    !Modification du profil de l’utilisateur
  • Enregistrez ;
    Il ne vous reste plus qu’à cliquer sur l’icône “Google” !bouton de connexion Google ( si l’email principale a pour nom de domaine ou terminaison : “gmail.com” ) ou sur l’icone “Yahoo” !bouton de connexion Yahoo ( si l’email principale a pour nom de domaine ou terminaison : “yahoo.fr ou yahoo.com” ) pour accéder à la plate forme.
