Définir l'adresse e-mail d'un utilisateur

23 janvier 2018

Attention : pour des raisons de sécurité, cette opération ne peut être effectuée qu’une seule fois par l’administrateur. Il est donc important d’éviter toute erreur dans l’écriture de l’adresse e-mail.

La procédure est la suivante :

  • aller à la section Console > Utilisateurs ;
  • cliquer sur l’icône pour afficher le profil complet de l’utilisateur concerné (AVOTO Juliana dans notre exemple) ;
  • dès que le profil est chargé, cliquer sur le bouton «Modifier» et remplir le champ EMAIL PRINCIPAL ;
  • terminer en cliquant sur le bouton Enregistrer.
