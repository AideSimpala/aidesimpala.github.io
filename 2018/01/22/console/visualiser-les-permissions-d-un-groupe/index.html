Visualiser les permissions d'un groupe d'utilisateurs

22 janvier 2018

Simpala vous offre la possiblité de visualiser l’ensemble des permissions octroyées à un groupe d’utilisateurs.
La démarche est simple :

  • aller à la section Console > Politique de sécurité ;
  • dans le panneau Gestionnaire des permissions, sélectionner un groupe ou une application (ou les deux) ;
  • la liste des permissions correspondantes s’affiche.

Les actions autorisées sont précédées d’une case cochée .
