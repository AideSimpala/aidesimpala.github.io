Changer mon mot de passe sur Simpala

22 janvier 2018

Dans la page d’identification, cliquer sur le lien “besoin d’aide ?”, à droite du bouton Connexion.
La fenêtre ci-dessous s’affiche :

  • Cliquer sur le bouton “Envoyer”.
  • Sous quelques minutes, pour devez recevoir un e-mail dans votre boite de reception.

Si au delà de 5 minutes, vous n’avez toujours rien reçu, pensez à verifier votre dossier Spam / courriers indésirables.
Dans le cas où le courrier de Simpala s’y trouverait, il convient tout d’abord de marquer comme n’etant pas un Spam.

  • Une fois que le message figure bien dans votre boite de reception, cliquer sur le bouton Activer mon compte.
    Vous êtes redirigé vers la page de définition de votre nouveau mot de passe.
  • Remplir le formulaire et cliquer sur le bouton de confirmation.

Si tout s’est bien passé, vous devriez être à présent automatiquement connecté et redirigé vers votre page d’accueil personnalisée.
