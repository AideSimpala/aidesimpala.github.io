Accéder rapidement à votre espace Simpala

22 janvier 2018

La plateforme Simpala vous offre la possibilité de vous identifier en un seul clic grace à notre integration avec Google et Yahoo.

Pour celà, il faut au préalable avoir renseigné l’adresse de l’utilisateur comme “Email principal” :
Cliquez sur “Utilisateurs” dans le menu latéral gauche ;

  1. Cliquez sur le bouton de présentation du profil de l’utilisateur choisit : ; par exemple l’utilisateur nommé “ONANA Sebastien”

La fenêtre suivante s’ouvre :

  1. Cliquez sur le bouton “Modifier” et saisissez l’email principale :
  2. Enregistrez ;
    Il ne vous reste plus qu’à cliquer sur l’icône “Google” ( si l’email principale a pour nom de domaine ou terminaison : “gmail.com” ) ou sur l’icone “Yahoo” ( si l’email principale a pour nom de domaine ou terminaison : “yahoo.fr ou yahoo.com” ) pour accéder à la plate forme.
