Saisir les notes

27 janvier 2018

Il est possible d’entrer les notes de vos élèves pour chaque séquence, par classe et par matière. Pour ce faire, vous pouvez vous servir d’un téléphone Android/Iphone, d’une tablette ou d’un ordinateur. La démarche est la suivante :

  • aller à la section Gradebook > Notes ;
  • choisir la classe concernée dans le menu déroulant GROUPE D’APPRENANTS ;
  • choisir la période dans le menu déroulant PÉRIODE ;
  • choisir la matière dans le menu déroulant COURS ;
  • cliquer sur l’onglet Saisie des résultats ;
  • plus bas dans la page, cliquer sur le bouton Commencer la saisie, puis sur Notes ;
  • remplir le formulaire avec les notes de chaque élève ;
  • remonter et cliquer sur le bouton Enregistrer.
