Configurer les paramètres financiers de son établissement

05 juin 2018

NB: Cette action n’est possible que si vous avez la permission suivante : Configurer les paramètres de l’application, dans Simpala Finance (en savoir plus).

Pour ce faire :

  • définir les frais de scolarité ;
  • définir les montants des frais de scolarités par classe ;
  • activer les rapports financiers journaliers.
