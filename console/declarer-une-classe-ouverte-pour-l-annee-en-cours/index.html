Déclarer une classe ouverte pour l'année en cours

27 janvier 2018

Une fois le groupe d’élèves créé (en savoir plus), il faut signaler au système qu’il s’agit d’une classe pour l’année scolaire en cours. La procédure est la suivante :

  • aller à la section Console > Groupes ;
  • dans la liste des groupes remontée, identifier le groupe en question puis cocher la case correspondante ;
  • au dessus de la liste de groupes, cliquer sur le bouton Définir les groupes sélectionnés comme classe pour la période académique X.

Vous devriez voir un message de confirmation sur fond vert si l’opération s’est bien déroulée.
