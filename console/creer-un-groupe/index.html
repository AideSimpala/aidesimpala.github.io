Créer un groupe personnalisé

27 janvier 2018

Par défaut, Simpala vous propose déjà des groupes classiques représentant les différentes parties prenantes au sein de votre organisation :

  • Élèves ;
  • Parents ;
  • Enseignants ;
  • Membre du personnel ;
  • Administrateur.

Toutefois, il est possible de créer des groupes spécifiques (Censeurs, Gestionnaires des finances, 6e A, 5e C). Pour ce faire :

  • aller à la section Console > Groupes ;
  • cliquer sur le bouton Créer un groupe situé à l’extrémité droite de la page ;
  • renseigner les informations sur le groupe ;
  • cliquer sur le bouton Enregistrer.
