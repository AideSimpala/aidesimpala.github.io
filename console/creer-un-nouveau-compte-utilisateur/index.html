Créer un nouveau compte utilisateur

22 février 2018

NB: Pour effectuer cette action, assurez-vous d’avoir la permission suivante : Ajouter de nouveaux utilisateurs et éditer leurs profils (en savoir plus)

Après vérification, suivez les étapes ci-dessous :

  • aller à la section Console > Utilisateurs ;
  • cliquer sur le bouton Ajouter et choisir la catégorie de l’utilisateur dans le menu déroulant qui s’affiche ;
  • éditer les informations de l’utilisateur ;
  • enfin, cliquer sur le bouton Enregistrer.
