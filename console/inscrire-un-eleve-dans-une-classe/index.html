Inscrire un élève dans une classe

21 février 2018

Pour inscrire un élève dans une classe avec Simpala, nous distinguons deux cas de figure :

Cas d’un élève ayant un dossier dans votre espace Simpala

  1. aller à la section Console > Utilisateurs ;
  2. effectuer la recherche de l’élève en tapant dans la barre de recherche les premières lettres de son nom, prénom ou matricule ;
  3. identifier l’élève dans les résultats de la recherche et ouvrir sa fiche de renseignements en cliquant sur la ligne correspondante ;
  4. une fois la page de profil de l’élève ouverte, cliquer sur l’onglet Groupes ;
  5. dans la section réservée aux inscriptions :
    • sélectionner la nouvelle classe de l’élève ;
    • si l’élève est Nouveau dans l’établissement, cocher la case correspondante (laisser vide pour un élève ayant été inscrit l’année précédente) ;
    • si l’élève redouble la classe (même en provenance d’un autre établissement), cocher la case Redoublant(e) (laisser vide pour un élève promu en classe supérieure) ;
    • cas d’établissements avec internat : si l’élève est Pensionnaire, cocher la case correspondante (laisser vide pour les externes) ;
    • cas de certaines crèches : si l’élève est scolarisé en mode Demi-journée, cocher la case correspondante (laisser vide pour ceux en mode journée complète) ;
  6. enfin, cliquer sur le bouton Enregistrer pour terminer.

Cas d’un nouvel élève (dossier inexistant dans votre espace Simpala)

  1. aller à la section Console > Utilisateurs ;
  2. cliquer sur le bouton Ajouter > Apprenant ;
  3. remplir les informations sur l’identité de l’élève ;
  4. dans la section réservée aux inscriptions :
    • si l’élève est Nouveau dans l’établissement, cocher la case correspondante (laisser vide pour un élève ayant été inscrit l’année précédente) ;
    • si l’élève redouble la classe (même en provenance d’un autre établissement), cocher la case Redoublant(e) (laisser vide pour un élève promu en classe supérieure) ;
    • cas d’établissements avec internat : si l’élève est Pensionnaire, cocher la case correspondante (laisser vide pour les externes) ;
    • cas de certaines crèches : si l’élève est scolarisé en mode Demi-journée, cocher la case correspondante (laisser vide pour ceux en mode journée complète) ;
  5. renseigner le(s) contact(s) de l’élève ;
  6. enfin, cliquer sur le bouton Enregistrer.
