Inscrire un élève dans une classe

21 février 2018

Pour inscrire un élève dans une classe avec Simpala, nous distinguons deux cas de figure :

Cas d’un élève ayant un dossier dans votre espace Simpala

  • aller à la section Console > Utilisateurs ;
  • effectuer la recherche de l’élève en tapant dans la barre de recherche les premières lettres de son nom, prénom ou matricule ;
  • identifier l’élève dans les résultats de la recherche et ouvrir sa fiche de renseignements en cliquant sur l’icône de la ligne correspondante ;
  • une fois la page de profil de l’élève ouverte, cliquer sur l’onglet Inscriptions ;
  • dans la section réservée aux inscriptions :
    • sélectionner la nouvelle classe de l’élève ;
    • si l’élève est redoublant, cocher la case correspondante ;
  • enfin, cliquer sur le bouton Enregistrer pour terminer.

Cas d’un nouvel élève (dossier inexistant dans votre espace Simpala)

  • aller à la section Console > Utilisateurs ;
  • cliquer sur le bouton Ajouter > Apprenant ;
  • remplir les informations sur l’identité de l’élève ;
  • dans la section réservée aux inscriptions :
    • sélectionner la nouvelle classe de l’élève ;
    • si l’élève est redoublant, cocher la case correspondante ;
  • renseigner le(s) contact(s) de l’élève ;
  • enfin, cliquer sur le bouton Enregistrer.
