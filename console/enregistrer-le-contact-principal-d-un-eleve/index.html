Enregistrer le contact principal d'un élève

27 janvier 2018

Avec la plateforme Simpala, vous pouvez enregistrer le contact principal de chaque élève. Pour y parvenir, suivez la démarche ci-dessous :

  • aller à la section Console > Utilisteurs ;
  • cliquer sur l’option de filtre pour une recherche ciblée ;
  • dans le menu déroulant GROUPE, choisir Apprenant ;
    la liste constituée de tous les élèves s’affiche :
  • identifier l’élève et cliquer sur le bouton de visualisation du profil de l’élève  ;
  • dans l’onglet contact, cliquer sur le bouton Modifier ;
  • editer le contact de l’élève (numéro d’un parent, d’un tuteur, etc) ;
  • cliquer sur le bouton Enregistrer.
